Quick Start — जल्दी शुरू करें

ये पेज उन लोगों के लिए है जो "जल्दी काम चलाना" चाहते हैं — सबसे जरूरी कदम, आसान भाषा में.

5-मिनट चेकलिस्ट (सबसे जरूरी)

  1. Lab बनाएं: VirtualBox पर VM बनाकर practice करें — Lab Setup
  2. पहला DC बनाएं: Server पर AD DS इंस्टॉल और Promote करें — AD Promote
  3. DNS चेक करें: क्लाइंट का DNS हमेशा DC का IP हो — DNS Guide
  4. DHCP स्कोप बनाएं: Clients को IP देने के लिए Scope बनाएं — DHCP Scope
  5. File share बनाएं: Data के लिए एक सरल शेयर और NTFS परमिशन सेट करें — File Server

अगर कुछ टूटे तो (तेज़ Diagnostic)

  • लॉगिन समस्या → Time sync और account lock देखें — Login Troubleshoot
  • नाम resolve नहीं हो raha → nslookup और DNS console देखें — DNS Validate
  • GPO नहीं लग रही → gpupdate / gpresult — GPO Validate

Recommended order (नया होना)

  1. Lab → AD Promote → DNS/DHCP → Users & GPO → File Server → Backup

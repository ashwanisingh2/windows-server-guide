Cheat Sheet — Quick Bullets

Essentials

  • DC पर static IP रखें
  • Client DNS = DC का IP
  • Production से पहले snapshot/backup लें

Quick Commands

  • DNS टेस्ट: nslookup <host>
  • Time sync: w32tm /resync
  • GPO apply: gpupdate /force
  • AD repl: repadmin /replsummary
  • DHCP stats: Get-DhcpServerv4ScopeStatistics

Setup Order (short)

  • Lab → AD promote → DNS → DHCP → Users/GPO → File share → Backup

Troubleshoot Fast

  • Login fail → time, account lock, klist
  • Name fail → client DNS, ipconfig /flushdns
  • Replication fail → dcdiag + repadmin /showrepl

Safety

  • GPO पहले pilot OU पर टेस्ट करें
  • DC IP बदलने से पहले console access रखें
  • Major change → maintenance window

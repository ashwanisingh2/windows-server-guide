सबसे पहले (Essentials)

  • Static IP on DC, client DNS = DC IP
  • Snapshot/backup before changes
  • Test commands on lab before prod

Setup Shortcuts (order)

  1. Lab → AD DS install → Promote DC
  2. DNS forward/reverse → Forwarders set
  3. DHCP scope + authorize
  4. Users/Groups → GPO baseline
  5. File share + NTFS perms → Shadow copies

Troubleshoot Fast

  • Login fail → check time, locked account, klist
  • Name fail → client DNS, ipconfig /flushdns, nslookup
  • Replication → dcdiag + repadmin /showrepl

Safety Rules

  • Change only in maintenance window for prod
  • Test GPO on pilot OU first
  • Never change DC IP without console access

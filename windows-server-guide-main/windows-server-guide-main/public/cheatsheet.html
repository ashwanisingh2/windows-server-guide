Windows Server Daily Cheat Sheet

Check first → root cause samjhein → phir change karein.

⚠️ Change se pehle

  • Backup/snapshot aur rollback plan ready.
  • Production change = approved maintenance window.
  • Remote IP change = console/iLO access ready.
  • GPO = pehle pilot OU par test.

🏥 AD health

DC quick health
dcdiag /q
Replication summary
repadmin /replsummary
Replication detail
repadmin /showrepl
Find a DC
nltest /dsgetdc:<domain>

🌐 Network + DNS

Full IP config
ipconfig /all
Resolve name
nslookup <host>
Clear DNS cache
ipconfig /flushdns
Test port
Test-NetConnection <server> -Port 53

👤 User + login

Locked users
Search-ADAccount -LockedOut
User details
Get-ADUser <user> -Properties *
Current identity
whoami /all
Kerberos tickets
klist

🧩 Group Policy

Apply policy
gpupdate /force
Quick result
gpresult /r
HTML report
gpresult /h gpo.html
Open GPMC
gpmc.msc

📡 DHCP

Renew client IP
ipconfig /release & ipconfig /renew
Scope status
Get-DhcpServerv4ScopeStatistics
Authorized servers
Get-DhcpServerInDC
DHCP service
Get-Service DHCPServer

📁 File + permissions

Group token
whoami /groups
NTFS permissions
icacls "<path>"
SMB shares
Get-SmbShare
Map/test share
net use \\<server>\<share>

⏱️ Time + services

Time status
w32tm /query /status
Resync time
w32tm /resync
Core services
Get-Service NTDS,DNS,Netlogon
Recent errors
Get-WinEvent -LogName System -MaxEvents 20

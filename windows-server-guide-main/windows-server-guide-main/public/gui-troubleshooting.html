🔧 Troubleshooting Master Guide

Windows Server ke common production issues — har ek ka symptom, diagnostic command, step-by-step fix, aur escalation trigger. Field engineer toolkit.

⚠️ Production par fix karne se pehle: snapshot lo, change window mein karo, console access ready rakho.

Troubleshooting Method (har issue par lagao)

Random commands mat chalao. Ye 5-step approach follow karo:

StepKya karo
1. DefineExact symptom + error message + Event ID note karo. "Kab se?" "Kya change hua?"
2. ScopeEk user/server ya sab? Pattern dhoondo (ek site, ek VLAN, ek OU?)
3. DiagnoseLogs (Event Viewer) + diagnostic commands. Layer by layer (network → DNS → service → app)
4. FixEk time par ek change. Test. Document.
5. VerifyIssue resolved confirm karo + side-effect check
💡 Event Viewer (eventvwr.msc) sabse pehla dost hai. System + Application + Security logs check karo.

1. Login / Authentication Issues

Symptom: User login nahi kar pa raha, "wrong password", account locked.

Causes: Account locked/expired, password expired, time skew >5 min (Kerberos fail), DC reachable nahi, group membership.

Get-ADUser amit.sharma -Properties LockedOut,PasswordExpired,Enabled
Search-ADAccount -LockedOut
w32tm /stripchart /computer:dc01 /samples:3
klist
nltest /dsgetdc:corp.company.local

Logs: DC par Event 4625 (failed logon), 4740 (lockout). Fix: Unlock-ADAccount amit.sharma; time fix w32tm /resync.

Escalate: Multiple users ek saath locked = possible brute-force attack → senior + security.

2. Group Policy Not Applying

Symptom: Policy setting client par reflect nahi ho rahi.

Causes: GPO not linked, security/WMI filtering, block inheritance, SYSVOL replication pending, client not domain-joined.

gpupdate /force
gpresult /r
gpresult /h C:\gpo.html
repadmin /showrepl

Fix: GPMC → Scope tab → security filtering check; inheritance check; SYSVOL replicate confirm.

Escalate: SYSVOL/DFSR replication broken across DCs → senior.

3. DHCP — IP Nahi Mil Rahi

Causes: Scope full, service stopped, server unauthorized, rogue DHCP, missing relay (ip-helper) on VLAN.

Get-DhcpServerInDC
Get-DhcpServerv4ScopeStatistics
Get-Service DHCPServer
ipconfig /release; ipconfig /renew

Fix: Authorize server, scope extend, switch par ip helper-address check.

Escalate: Rogue DHCP detected → network + security team.

4. DNS — Name Resolution Fail

Causes: Wrong DNS on client, stale records, forwarder down, zone not replicating.

nslookup corp.company.local
Resolve-DnsName fs01.corp.company.local
dcdiag /test:dns
ipconfig /flushdns

Fix: Client DNS = DC IP; stale record delete; forwarder fix; scavenging review.

⚠️ Client ka DNS kabhi public (8.8.8.8) par mat rakho domain mein — DC ka IP hona chahiye.

5. File / Share Access Denied

Causes: NTFS vs Share conflict (most restrictive wins), explicit Deny, group change (re-login needed), token bloat, ABE.

whoami /groups
icacls "E:\Data\Finance"
net use \\FS01\Finance

Fix: Security → Advanced → Effective Access tab; re-login for fresh token; conflicting Deny remove.

Escalate: Token size issue (user 1000+ groups mein) → AD group cleanup project.

6. AD Replication Fail

Causes: DNS fail (most common), time skew, firewall (RPC/135), KCC issue, USN rollback, lingering objects.

repadmin /replsummary
repadmin /showrepl
dcdiag /test:replications
portqry -n dc02 -p both -e 135
⚠️ USN rollback (DC galat snapshot se restore) = replication permanently break → DC rebuild. Escalate immediately.

7. DC Crash / Restore

Recoverable: DSRM boot → system state restore. Objects deleted: AD Recycle Bin restore. Unrecoverable: metadata cleanup + seize FSMO.

Get-ADObject -Filter 'isDeleted -eq $true' -IncludeDeletedObjects | Restore-ADObject
Move-ADDirectoryServerOperationMasterRole -Identity DC02 -OperationMasterRole 0,1,2,3,4 -Force
Escalate: FSMO seize / authoritative restore = senior approval mandatory. Seize ke baad purana DC network par mat lao.

8. Performance / Server Slow

Symptom: High CPU/RAM/disk, slow logon, app timeout.

Get-Process | Sort CPU -Descending | Select -First 10
Get-Counter "\Memory\Available MBytes"
Get-Counter "\PhysicalDisk(_Total)\% Disk Time"
Get-Counter "\Processor(_Total)\% Processor Time"

Fix: Top process identify; Task Manager / Resource Monitor (resmon); memory leak service restart; antivirus scan check.

💡 Baseline pehle se rakho (normal CPU/RAM) — tabhi pata chalega "slow" kya hai.

9. Boot Failure / BSOD

Causes: Bad driver/update, disk corruption, failed AD service.

:: Safe Mode / DSRM boot ke baad
sfc /scannow
DISM /Online /Cleanup-Image /RestoreHealth
chkdsk C: /f

Fix: Last Known Good / Safe Mode; recent update uninstall; bcdedit se boot fix; BSOD code note karo (memory dump analyze).

Escalate: DC boot nahi ho raha aur recovery fail → senior (DSRM restore / rebuild).

10. Windows Update Fail

Symptom: Update stuck, error code, reboot loop.

:: Update service reset
net stop wuauserv
net stop bits
ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
net start wuauserv
net start bits

Fix: SoftwareDistribution folder reset; disk space check; specific KB manual download; WSUS sync check.

11. Service Crash / Won't Start

Get-Service | Where Status -eq "Stopped" | Where StartType -eq "Automatic"
Get-WinEvent -LogName System -MaxEvents 50 | Where LevelDisplayName -eq "Error"
Start-Service -Name "ServiceName"

Fix: Event Viewer mein crash reason; dependency service check (services.msc → Dependencies tab); recovery action set (restart on failure).

12. Time Sync Issue (Kerberos killer)

Symptom: Login fail, replication fail, >5 min skew.

w32tm /query /status
w32tm /stripchart /computer:dc01 /samples:5
w32tm /resync /force
:: PDC ko external NTP point karo
w32tm /config /manualpeerlist:"pool.ntp.org" /syncfromflags:manual /update
⚠️ PDC Emulator (FSMO) authoritative time source hai — use external NTP se sync karo, baaki sab DC/clients usse.

13. Disk Full / Storage

Get-Volume
Get-ChildItem C:\ -Recurse -EA SilentlyContinue | Sort Length -Descending | Select -First 20 FullName,Length

Fix: Temp/log cleanup, cleanmgr, old WinSxS cleanup (DISM /Online /Cleanup-Image /StartComponentCleanup), log rotation, disk extend.

⚠️ DC par C: full = NTDS/SYSVOL fail = domain down. Disk space alert pehle se setup karo.

14. Network / Connectivity

Test-NetConnection dc01 -Port 389
Get-NetIPConfiguration
ipconfig /all
Tracert 8.8.8.8

Fix: IP/gateway/DNS verify; NIC teaming status; firewall rule check; cable/switch port; VLAN tag.

15. Certificate / PKI

Symptom: HTTPS warning, VPN/Wi-Fi auth fail, cert expired.

certutil -verify -urlfetch cert.cer
Get-ChildItem Cert:\LocalMachine\My | Select Subject,NotAfter
certutil -crl

Fix: Expired cert renew/re-enroll; CRL distribution point reachable; auto-enrollment GPO; root CA trust check.

⚠️ CRL inaccessible = sab cert validation fail. CDP hamesha reachable rakho.

16. Printing Issues

Get-Service Spooler
Restart-Service Spooler
Get-Printer
Get-PrintJob -PrinterName "HQ-Finance-HP"

Fix: Spooler restart; stuck job clear (C:\Windows\System32\spool\PRINTERS empty karo); driver re-install; isolation mode for bad driver.

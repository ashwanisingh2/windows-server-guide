👥 Users, OU & Group Policy (GUI)

ADUC se users/OU/groups, GPMC se policies — click-by-click.

📌 Is page ka context
KYA
OU structure, users, groups, delegation (ADUC) aur Group Policy (GPMC) banana.
KYU
Role-based access + centralized policy. Bina iske 500 users aur 100 PCs manage impossible.
KAISE
ADUC se OU/user/group → delegate control → GPMC se GPO create + link. Neeche steps.

Open ADUC

Server Manager → Tools → Active Directory Users and Computers  ya  dsa.msc

Create OU

  1. Right-click domain corp.company.local → New → Organizational Unit
  2. Name: CORP-HQ → OK (sub-OUs banao: Users, Computers, Groups)
💡 Geographic + Department structure rakho. Default Users/Computers container par GPO nahi lagti.

Create User

  1. Right-click target OU → New → User
  2. First/Last name, Logon name: amit.sharma → Next
  3. Password set → ✅ "User must change password at next logon" → Next → Finish

Create Group

  1. Right-click OU → New → Group
  2. Name: SG-Finance → Scope: Global → Type: Security → OK
  3. Member add: right-click user → Add to a group → SG-Finance
💡 AGDLP: Accounts → Global group → Domain Local group → Permission.

Delegate Control (helpdesk ko rights)

  1. Right-click OU → Delegate Control → Next
  2. Add helpdesk group → Next
  3. ✅ "Reset user passwords and force change at next logon" → Next → Finish

Open Group Policy Management (GPMC)

Tools → Group Policy Management  ya  gpmc.msc

Create & Link GPO

  1. GPMC → right-click target OU → Create a GPO in this domain, and Link it here
  2. Name: Baseline-Security → OK
  3. Right-click GPO → Edit (Group Policy Editor khulega)
  4. Set policies (table neeche)
PolicyPath
Password length 14Computer → Windows Settings → Security → Account Policies
Account lockout 5/15minSame
USB blockComputer → Admin Templates → System → Removable Storage
Drive mappingUser → Preferences → Drive Maps
⚠️ USB block / AppLocker domain-wide directly mat lagao. Pehle pilot OU (5-10 machines), phir rollout.

Validate

  • Client par CMD: gpupdate /force
  • gpresult /r → applied GPOs list dekho
  • gpresult /h report.html → detailed report

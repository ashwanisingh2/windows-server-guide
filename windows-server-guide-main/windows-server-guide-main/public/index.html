REMEMBER

6 Golden Rules

1. DC = Static IPDomain Controller par dynamic IP mat rakhein.
2. Client DNS = DC IPClient par public DNS dene se domain issues aayenge.
3. Backup firstChange se pehle snapshot/backup aur rollback plan.
4. Pilot firstGPO ko pehle 1–5 test computers par lagayein.
5. Check time5 minute se zyada skew Kerberos login tod sakta hai.
6. Validate every step“Command chal gayi” ka matlab “setup sahi hai” nahi.

STOP AND ESCALATE

Senior ko kab bulana hai?

  • Multiple users ek saath lock ho rahe hain
  • AD replication/SYSVOL broken hai
  • FSMO seize, USN rollback ya DC restore karna hai
  • Unexpected Domain Admin ya security breach dikhe

Koi result nahi mila

DNS, DHCP, login, GPO, share ya replication search karke dekhein.

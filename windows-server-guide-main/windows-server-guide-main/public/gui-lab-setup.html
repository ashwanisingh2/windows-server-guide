🧪 Lab Setup Guide (VirtualBox)

Apne PC par free practice lab banao — Windows Server VM. Beginner-friendly, step-by-step.

📌 Is page ka context
KYA
VirtualBox par Windows Server ka virtual lab banana, jisme website ke saare steps practice kar sako.
KYU
Padhna ≠ karna. Real practice se hi confidence aata hai. Lab mein galti karna safe hai.
KAISE
VirtualBox install → Server ISO → VM banao → install → snapshot → practice. Neeche steps.

Step 0: PC Ready Hai? (Check)

RequirementMinimumBetter
RAM8 GB16 GB

💡 Virtualization check: Task Manager → Performance → CPU → niche "Virtualization: Enabled" dikhna chahiye. Agar Disabled → BIOS mein VT-x/AMD-V enable karo.

Step 1: Install VirtualBox

  1. Browser → virtualbox.org → Downloads → Windows hosts
  2. Installer run karo → Next-Next-Finish (default settings theek hain)
  3. Network warning aaye to → Yes

Step 2: Windows Server ISO Download (Free 180 days)

  1. Google: "Windows Server 2022 Evaluation" → Microsoft Evaluation Center
  2. ISO download karo (free, 180-day full trial)
  3. (Optional) Windows 10/11 ISO bhi — client machine ke liye
⚠️ Evaluation = full features, sirf 180 din. Lab ke liye perfect. Rearm karke extend bhi ho sakta hai.

Step 3: VM Banao

  1. VirtualBox → New
  2. Name: DC01 · Type: Microsoft Windows · Version: Windows 2022 (64-bit)
  3. RAM: 4096 MB (4 GB)
  4. Disk: Create virtual disk → VDI → Dynamically allocated → 60 GB
  5. VM select → Settings → Storage → empty CD icon → Server ISO attach
  6. Settings → Network → Adapter 1 → Internal Network (client se connect ke liye) ya Bridged

Step 4: Windows Server Install

  1. VM Start
  2. Language → Next → Install Now
  3. Edition: Windows Server 2022 Datacenter (Desktop Experience) — GUI ke liye zaroori
  4. Custom install → disk select → Next
  5. Install hone do → reboot → Administrator password set karo
  6. Login → Server ready! ✅
💡 Install ke baad VirtualBox menu → Devices → Insert Guest Additions CD → install karo (mouse/screen smooth ho jaata hai).

Step 5: Snapshot Lena (sabse zaroori habit)

  1. Har bade step se pehle: VirtualBox → right-click VM → Snapshot → Take
  2. Naam do: e.g. "Fresh install", "Before AD", "After AD"
  3. Kuch galat ho → us snapshot par Restore → wapas us point par
⚠️ Snapshot = lab ka sabse bada fayda. Bindaas todo, fix karo, ya restore karo. Real server par ye luxury nahi hota.

Step 6 (Optional): Client VM

Ek aur VM banao Windows 10/11 ka (same Internal Network). Isse domain join karo — taaki GPO, login, share access real test kar sako.

⚙️ Advanced Roles (GUI)

Print Server, Hyper-V, Backup, PKI — Server Manager se click-by-click.

📌 Is page ka context
KYA
Print Server, Hyper-V, Backup/DR aur PKI roles install + configure karna.
KYU
Centralized printing, virtualization, disaster recovery aur internal certificates — scalability + security.
KAISE
Add Roles and Features se har role install → respective console se configure. Neeche steps.

Print Server

  1. Add Roles and Features → Server Roles → ✅ Print and Document Services → Install
  2. Tools → Print Management
  3. Print Servers → server → right-click Printers → Add Printer → TCP/IP → printer IP daalo
  4. Driver select → Share name do → Finish
💡 GPO se printer deploy: Print Management → right-click printer → Deploy with Group Policy.

Hyper-V

  1. Add Roles and Features → ✅ Hyper-V → NIC select → Install → reboot
  2. Tools → Hyper-V Manager
  3. Right-click server → Virtual Switch Manager → New → External/Internal/Private → Apply
  4. Right-click server → New → Virtual Machine → wizard follow (Gen2, memory, VHD, switch)
⚠️ Checkpoint ≠ Backup. Production DB/DC par lambe checkpoints mat rakho.

Backup & DR (Windows Server Backup)

  1. Add Roles and Features → Features → ✅ Windows Server Backup → Install
  2. Tools → Windows Server Backup → Local Backup
  3. Right-click → Backup Schedule → wizard
  4. ✅ Full server + ✅ System State (AD recovery ke liye CRITICAL) → target disk → time set → Finish
MetricDefine
RPOMax data loss (e.g. 24h)
RTOMax downtime (e.g. 4h)
Restore drillMonthly test
⚠️ Tested backup = backup. Untested = hope. 3-2-1 rule: 3 copies, 2 media, 1 offsite.

Certificate Services (PKI)

  1. Add Roles and Features → ✅ Active Directory Certificate Services → ✅ Certification Authority + Web Enrollment → Install
  2. Yellow flag → Configure Active Directory Certificate Services
  3. Enterprise CA → Root CA → New private key → SHA256, 4096-bit → validity 10 years → Configure
⚠️ Production best practice: Offline Root CA + Online Subordinate (Issuing) CA. CA name baad mein change nahi hota — pehle final karo.
